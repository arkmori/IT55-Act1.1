How to write an essay the way I do

  1. Develop an opinion based on the topic.
  2. Write a draft. It does not need to be perfect.
  3. Rewrite the draft. Switch sentences or paragraphs with each other if need be. Do not be afraid to add or remove ideas.
  4. Finish the essay. Don't worry too much if you're attending an essay-writing or related competition.
